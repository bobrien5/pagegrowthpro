How to get a Facebook Page Access Token
  1. Go to developers.facebook.com and create an App (type: Business)
  2. Open Graph API Explorer, select your App
  3. Click "Generate Access Token", grant pages_manage_posts and pages_read_engagement
  4. Select your Page from the "User or Page" dropdown, copy the token
  5. Exchange for long-lived token: GET /oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=TOKEN
  6. Get permanent Page token: GET /me/accounts?access_token=LONG_LIVED_TOKEN
  7. The Page token from step 6 never expires. Paste it above.
